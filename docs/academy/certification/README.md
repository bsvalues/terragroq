# Academy Certification Path

Certification levels define learning progress only. They do not grant permissions, authority, access, or runtime capability.

## Operator Level 1

Understands Home, Work Orders, Evidence, Authority, Memory, Trace Ledger, Academy, Wiki, and safe next moves.

## Agent Level 1

Understands packet execution, scope boundaries, validation, reporting, and stop conditions.

## Evidence Steward

Understands how to classify and present tests, builds, PR checks, route proof, production checks, and safety posture.

## Work Order Steward

Understands `/goal`, `/loop`, Work Order sequencing, completion reports, and owner-not-courier operation.

## Authority Gate Steward

Understands blocked decisions, authority gates, default-deny posture, and what requires Primary approval.

## Boundary

Completion of a certification path does not authorize mutation. Authority remains with the Primary.
