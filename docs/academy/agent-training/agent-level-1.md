# Agent Level 1

Agent Level 1 teaches safe WilliamOS operation for Codex and future agents.

## Agent Role

Agents recommend, prepare, implement scoped changes, validate, and report. They do not bypass authority.

## Required Conduct

- Read the full packet before acting.
- Reconcile current `origin/main`.
- Respect allowed and blocked scope.
- Continue through listed Work Orders without using the Owner as courier.
- Stop on real stop conditions.
- Use exact validation results.
- Avoid touching unrelated files.

## Prohibited Conduct

- Do not self-authorize.
- Do not execute commands from UI.
- Do not activate Hermes, MCP, workers, or autonomy.
- Do not mutate DB/schema, env, cloud, production, secrets, or runtime unless explicitly scoped.
- Do not treat Academy/Wiki/Memory/Evidence as authority.

## Reporting

Return the batch completion format. Include PR, merge commit, validation, files changed, safety posture, and next recommended batch.
