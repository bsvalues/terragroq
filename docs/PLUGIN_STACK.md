# Recommended Plugin and Tool Stack

## Obsidian Core

- Backlinks
- Graph View
- Canvas
- Daily Notes
- Templates

## Obsidian Community Plugins

### Essential

- Dataview: dashboards and queryable notes
- Templater: structured note creation
- Tasks: light action tracking
- Calendar: daily/weekly rhythm
- Periodic Notes: reviews
- QuickAdd: fast capture
- Linter: metadata and formatting hygiene
- Omnisearch: stronger search
- Excalidraw: visual systems thinking

### Later

- Smart Connections or equivalent semantic search
- Obsidian Git
- Advanced Tables
- Zotero integration
- Readwise/Reader integration

## CLI Tools

- git: memory ledger
- rg/ripgrep: fast search
- fd: fast file finding
- fzf: fuzzy navigation
- bat: readable previews
- jq: Graphify JSON analysis
- yq: YAML/frontmatter analysis
- pandoc: export Markdown to DOCX/PDF/HTML
- just: command runner
- python: scripts
- Graphify: graph extraction and analysis

## AI/MCP Tools

- Claude Desktop or Claude Code for long synthesis
- Codex/ChatGPT for strategy and drafting
- Obsidian MCP server later
- Ollama later for local/private summarization
