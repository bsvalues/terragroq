# WilliamOS Operating Rhythm

## Daily

```bash
python scripts/william.py today
```

Use the daily command note to capture:

- Today’s center
- Must move
- Waiting on
- One hard decision
- One thing to learn
- One thing to explain better
- End-of-day truth

## Capture anytime

```bash
python scripts/william.py inbox "Your thought here"
```

Capture first. Organize later.

## Weekly

```bash
python scripts/william.py weekly
python scripts/william.py stale-decisions
python scripts/william.py orphans
python scripts/william.py check
```

Then commit:

```bash
git add .
git commit -m "Weekly WilliamOS review YYYY-W##"
```

## Monthly

Run Graphify and create a graph review note.

Questions:

- What became doctrine?
- What became a project?
- What became noise?
- What did I finally understand?
- What should I stop thinking about?
