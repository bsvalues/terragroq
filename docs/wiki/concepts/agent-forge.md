# Agent Forge

Agent Forge is the WilliamOS capability preparation surface.

It organizes skill ideas, proposed capabilities, quarantine state, risk score,
permission boundaries, and activation review requirements before anything can
be treated as a live capability.

## What It Is

- A place to describe proposed skills and capabilities.
- A quarantine model for unreviewed or risky capability ideas.
- A governance surface for risk, evidence, and permission review.
- A bridge to Hermes only after separate authority gates.

## What It Is Not

- Not an executable skill loader.
- Not a dependency installer.
- Not a tool-calling runtime.
- Not a permission grant.
- Not an autonomy system.

Agent Forge can prepare capability knowledge. It cannot activate it.
